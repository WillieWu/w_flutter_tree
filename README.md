# w_flutter_tree

*** 一个简单的树形图展示库 ***

## Getting Started

``` dart
import 'package:w_flutter_tree/w_flutter_tree.dart';

/// 配置树形图样式
CustomTreeNodeStyle({
    this.direction = Axis.horizontal,
    this.mainAxisLeadingWidth = 30,
    this.mainAxisTrailingWidth = 30,
    this.crossAxisSpace = 20,
    this.strokeColor = const Color(0xFFB9B9B9),
    this.strokeWidth = 1,
  });

```
